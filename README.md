# VoltEye Mobile

Esqueleto de aplicativo mobile feito com React Native e Expo para acompanhamento de consumo de energia.

## Como rodar

```bash
npm install
npm start
```

Depois, abra no Expo Go ou use os atalhos do terminal para Android, iOS ou web.

## Estrutura

- `App.tsx`: entrada do aplicativo e controle simples de abas.
- `src/screens`: telas iniciais do app.
- `src/components`: componentes visuais reutilizaveis.
- `src/theme`: cores, espacamentos e estilos compartilhados.
- `src/data`: dados mockados para prototipacao.
